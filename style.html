Accessibility

Guidelines
  • Text contrast ≥ 4.5:1 on bg.
  • Touch target ≥ 44×44px; main capture ≥ 64px.
  • Announce ratio, burst and GPS state to screen readers.
  • Respect dynamic type up to ~120% without clipping primary actions.
This style guide is intentionally semantic and componentized so html.to.design imports cleanly. Promote recurring specimens (Button, Chip, Field, Card, Sheet, Toast) to components and add variant props.
